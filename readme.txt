Duplizieren von Beiträgen und Seiten.

== Description ==

Dieses Plugin ermöglicht das einfache und schnelle duplizieren von Seiten und Beiträgen in Ihrer WordPress Instanz. 

So funktioniert's:

1. In den Beiträgen und Seiten erscheint unter dem Beitragsname/Seitennamen ein Link zum 'Duplizieren'.

1. In 'Edit Posts'/'Edit Pages', you can click on 'Clone' link below the post/page title: this will immediately create a copy and return to the list.

2. Durch anklicken von mehreren Seiten, können Beiträge und Seiten gleichzeitig dupliziert werden.

3. In der Editieren Ansicht eines Beitrages oder einer Seite erscheint unter dem "Abbrechen/In den Papierkorb legen" der 'Duplizieren' Link.

4. Während ein Beitrag oder eine Seite von einem anderen Nutzer bearbeitet wird, können Sie via "Diese Seite als Entwurf duplizieren" eine Kopie der Seite ablegen und bearbeiten.

== Installation ==

Nutzen Sie die Suche im WordPress Plugin Verzeichnis und suchen nach 'Duplizieren von Beiträgen und Seiten', oder laden Sie das Plugin auf der offiziellen WordPress Plugin Seite herunter und laden dies manuell in Ihre WordPress Instanz hoch.
